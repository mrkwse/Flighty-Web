Flight School

Controlling planes in the game couldn't be any easier for users, already-implemented point-and-click functionality works alongside manual keyboard control, providing the user with the method optimal for them. The implementation of multiple control methods means a solid foundation to adapt Flighty to use any interface that you could possibly dream of using.
