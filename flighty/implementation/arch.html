Freshly ground. Get your morning fix of Java goodness.

Java was an obvious choice of object-oriented-langauge. It produces simple elegant code, supplemented by a range of libraries.

Equality at last. Idyllic multiplatform wonder.

The use of Java means a vast myriad of possible users and platforms, with the possibility of expanding Flighty to Android and other mobile platforms.

There's an app for that. Precompiled for Windows and Linux, so you don't have to.

Flighty comes compiled as a native application for Windows and Linux, and is easily executable as a .jar on Mac OS X. Less time spent banging your head against a keyboard and more time spent banging planes into one another.
